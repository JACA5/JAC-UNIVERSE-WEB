# Jorge Casas Universe
## jorgeacasas.com — Página web personal

Sistema solar interactivo con navegación inmersiva. Dark/gold aesthetic, responsive, accesible WCAG 2.1 AA.

### Características

- **Sistema solar navegable** — 11 planetas, 3 órbitas, cada uno representa una sección (Acerca, Expertise, Blog, Arsenal IA, Lab IA, Cursos, Servicios, Membresía, Streaming, Directorio, Contacto)
- **Directorio curado** — 107 herramientas de IA categorizadas (LLMs, Agentes, Video, Imagen, Audio, Educación, etc.)
- **Testimonios verificados** — De empresas, universidades y estudiantes
- **Generative art** — Flow field con partículas vivas, semilla determinística "JORGE"
- **Sistema de temas** — Dark, Light, Alto Contraste con toggle
- **Accesibilidad** — WCAG 2.1 AA, skip nav, focus visible, aria labels, navegación por teclado
- **Micro-interacciones** — Burst de partículas en clics, scroll reveal, orbit pulse, cursor glow

### Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Cormorant Garamond, Space Mono, Syne)
- Canvas API (starfield + flow field generativo)
- IntersectionObserver (scroll animations)
- LocalStorage (persistencia de temas)
- Sin dependencias externas — single-file deployment

### Cómo usar

1. Descarga `index.html`
2. Abre en navegador (funciona sin servidor)
3. O deploy a cualquier hosting: Vercel, Netlify, Hostinger

### Deploy automático

Este repo está conectado a Vercel. Cada push a `main` redeploya automáticamente.

URL en vivo: https://jac-universe-web.vercel.app/
